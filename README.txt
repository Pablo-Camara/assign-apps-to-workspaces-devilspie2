# Assign Apps to Workspaces with devilspie2

Requirements:
1 - Install Devilspie2:

If On Linux Mint ( in my case Tricia ) run in terminal:
sudo apt install devilspie2

or for more information check devilspie2 download website:
https://www.nongnu.org/devilspie2/

2 - Edit "configs/workspace_assigner.lua"  ( adapt to your preferences, use examples in code )

3 - Run devilspie2: 

sh devilspy.sh



Note:
You can create a launcher in your linux setup for example so with one click everything goes
to the right workspace;
